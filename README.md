# 🎬 BookMyShow Clone by Kamal

A full-stack BookMyShow clone built using the MERN stack with complete user and partner functionalities.

## 🚀 Features

### 👤 Authentication
- User Signup & Signin
- Forgot Password (Email Reset via Mailgun)

### 🎟 User Features
- Browse Movies
- Select Show Time
- Choose Seats
- Book Tickets
- Secure Payments using Stripe (Test Mode)

### 🏢 Partner Features
- Register as Partner
- Create Theatres
- Add Screens inside Theatres
- Publish Movies on the Platform

## 🛠 Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Email Service: Mailgun
- Payments: Stripe (Test Mode)

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies:
   npm install

3. Create `.env` file in server:
   - MONGO_URI=
   - JWT_SECRET=
   - STRIPE_SECRET_KEY=
   - MAILGUN_API_KEY=

4. Run backend:
   npm run server

5. Run frontend:
   npm run client
